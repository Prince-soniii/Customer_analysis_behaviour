📊 Customer Shopping Behavior Analysis
🔍 Overview

This project analyzes customer shopping behavior using transactional data to uncover insights into spending patterns, customer segments, and product preferences. The goal is to support data-driven business decisions using Python, SQL, and Power BI.

📂 GitHub Repository:
https://github.com/Prince-soniii/Customer_analysis_behaviour

👤 Author: Prince Soni
🔗 LinkedIn: https://www.linkedin.com/in/prince-soni-284181292/

📁 Dataset
Total Records: 3,900 transactions
Features: 18 columns including:
Customer demographics (Age, Gender, Location)
Purchase details (Item, Category, Amount, Season)
Behavior data (Discount, Subscription, Frequency, Rating)
Missing Data handled in Review Rating column
🛠️ Tools & Technologies
Python (Pandas, NumPy, Matplotlib, Seaborn)
SQL (PostgreSQL / MySQL / SQL Server)
Power BI
Gamma (Presentation)

⚙️ Project Workflow
1. Data Loading & Exploration
Loaded dataset using Pandas
Used .info() and .describe()
2. Data Cleaning
Handled missing values
Removed redundant columns
Standardized column names
Created age_group feature
3. Exploratory Data Analysis (EDA)
Data distribution & trends
Visual insights
4. SQL Analysis
Revenue by gender
High-spending discount users
Top-rated products
Customer segmentation
Revenue by age group

📊 Dashboard
Interactive Power BI dashboard
KPIs:
Total Customers: 3.9K
Avg Purchase: $59.76
Avg Rating: 3.75
Filters: Category, Gender, Shipping Type

📈 Results & Insights
Higher revenue from male customers
Discounts don’t always reduce spending
Loyal customers are highest segment
Young adults generate highest revenue
